<p><strong>Postgres sql :&nbsp;</strong></p>

<p>command for importing db sql file to local database</p>

<div data-darkreader-inline-bgcolor="" data-darkreader-inline-bgimage="" data-darkreader-inline-border-bottom="" data-darkreader-inline-border-left="" data-darkreader-inline-border-right="" data-darkreader-inline-border-top="" style="background: rgb(238, 238, 238); border: 1px solid rgb(204, 204, 204); padding: 5px 10px; --darkreader-inline-bgimage: initial; --darkreader-inline-bgcolor:#302e2b; --darkreader-inline-border-top:#4c4b46; --darkreader-inline-border-right:#4c4b46; --darkreader-inline-border-bottom:#4c4b46; --darkreader-inline-border-left:#4c4b46;">psql -h localhost -p 5433 --username=postgres <strong>&lt;kbr_db&gt;</strong>&nbsp; &lt;&nbsp; file_path&nbsp;</div>

<p><strong>note:&nbsp; &lt;kbr_db&gt;&nbsp;&nbsp;&nbsp;must be created previously<br />
<br />
eg:<br />
psql -h localhost -p 5433 --username=postgres kbr_db &lt; /home/sakthi/Downloads/kbr_db.sql</strong></p>

<hr />
<p><br />
postgresql commands :<br />
&nbsp;&nbsp; &nbsp;\l &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp; - to list all the databases<br />
&nbsp;&nbsp; &nbsp;\c database name - to connect to the database<br />
&nbsp;&nbsp; &nbsp;\dt &nbsp;&nbsp; &nbsp;&nbsp;&nbsp; &nbsp; - to list the database contents<br />
&nbsp;&nbsp; &nbsp;create database databaseName&nbsp;<br />
&nbsp;&nbsp; &nbsp;</p>
